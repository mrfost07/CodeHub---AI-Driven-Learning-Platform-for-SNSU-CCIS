## 🎨 **Customization Tips**

### **Want to Change Colors?**

Edit `frontend/src/pages/Home.tsx`:

```typescript
// Change main gradient:
background: 'linear-gradient(135deg, #YOUR_COLOR 0%, #YOUR_COLOR2 100%)'

// Change accent color:
background: 'linear-gradient(45deg, #YOUR_ACCENT 30%, #YOUR_ACCENT2 90%)'
```
